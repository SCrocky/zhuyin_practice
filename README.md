# zhuyin/bopomofo practice

A repo to test your knowledge on the Zhuyin characters

### Requirements

You must be in a python environment and have all dependencies installed.

Dependencies can be installed with `pip install -r requirements.txt`

### Execute

To run the scrip please run the `launch_quiz.py` file:

```
python launch_quiz.py
```

You can also use the `--nb` and `--guess` options.
